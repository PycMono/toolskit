📖 常见问题

什么是 camelCase？

驼峰命名法：第一个单词小写，后续单词首字母大写，如 myVariableName。常用于 JavaScript 变量和函数命名。

snake_case 和 kebab-case 有什么区别？

snake_case 用下划线分隔，常用于 Python 和数据库；kebab-case 用连字符分隔，常用于 URL 和 CSS 类名。

PascalCase 和 camelCase 的区别？

PascalCase（大驼峰）每个单词首字母都大写，如 MyClassName；camelCase（小驼峰）第一个单词小写，如 myClassName。PascalCase 常用于类名。
