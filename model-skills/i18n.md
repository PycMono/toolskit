## 二、后端规范

### 2.1 翻译加载器

```go
// internal/i18n/i18n.go

package i18n

import (
    "encoding/json"
    "fmt"
    "os"
    "strings"
    "sync"
)

var (
    translations = map[string]map[string]interface{}{}
    mu           sync.RWMutex
)

// Load 启动时调用，加载全部语言文件
func Load(dir string) error {
    for _, lang := range []string{"zh", "en"} {
        path := fmt.Sprintf("%s/%s.json", dir, lang)
        data, err := os.ReadFile(path)
        if err != nil {
            return fmt.Errorf("i18n: failed to read %s: %w", path, err)
        }
        var m map[string]interface{}
        if err := json.Unmarshal(data, &m); err != nil {
            return fmt.Errorf("i18n: failed to parse %s: %w", path, err)
        }
        mu.Lock()
        translations[lang] = m
        mu.Unlock()
    }
    return nil
}

// T 根据语言和点号分隔的 Key 返回翻译值
// 找不到时回退顺序：指定语言 → zh → Key 本身
func T(lang, key string) string {
    mu.RLock()
    defer mu.RUnlock()

    val := lookup(translations[lang], key)
    if val != "" {
        return val
    }
    // 回退到中文
    val = lookup(translations["zh"], key)
    if val != "" {
        return val
    }
    return key // 最终回退：返回 Key 本身，方便排查遗漏
}

func lookup(m map[string]interface{}, key string) string {
    if m == nil {
        return ""
    }
    parts := strings.SplitN(key, ".", 2)
    val, ok := m[parts[0]]
    if !ok {
        return ""
    }
    if len(parts) == 1 {
        if s, ok := val.(string); ok {
            return s
        }
        return ""
    }
    if sub, ok := val.(map[string]interface{}); ok {
        return lookup(sub, parts[1])
    }
    return ""
}
```

### 2.2 中间件

语言检测优先级：`URL ?lang=` → `Cookie lang` → `Accept-Language Header` → 默认 `zh`

```go
// internal/middleware/i18n.go

package middleware

import (
    "net/http"
    "strings"

    "github.com/gin-gonic/gin"
)

const (
    LangZH      = "zh"
    LangEN      = "en"
    LangCookie  = "lang"
    LangCtxKey  = "lang"
    DefaultLang = LangZH
)

func I18n() gin.HandlerFunc {
    return func(c *gin.Context) {
        lang := detectLang(c)
        c.Set(LangCtxKey, lang)

        // 写入 Cookie（30 天），让下次访问保持语言偏好
        if c.Query("lang") != "" {
            c.SetCookie(LangCookie, lang, 60*60*24*30, "/", "", false, false)
        }

        c.Next()
    }
}

func detectLang(c *gin.Context) string {
    // 1. URL 参数优先（?lang=en）
    if q := c.Query("lang"); isValidLang(q) {
        return q
    }
    // 2. Cookie
    if ck, err := c.Cookie(LangCookie); err == nil && isValidLang(ck) {
        return ck
    }
    // 3. Accept-Language Header
    if al := c.GetHeader("Accept-Language"); al != "" {
        if strings.HasPrefix(strings.ToLower(al), "zh") {
            return LangZH
        }
        return LangEN
    }
    return DefaultLang
}

func isValidLang(lang string) bool {
    return lang == LangZH || lang == LangEN
}
```

### 2.3 注册翻译函数到模板（main.go / router.go）

```go
// main.go 或 router.go 初始化阶段

import (
    "html/template"
    "yourproject/internal/i18n"
)

func initTemplates(cfg *config.Config) *template.Template {
    // 启动时加载翻译文件
    if err := i18n.Load("i18n"); err != nil {
        log.Fatalf("i18n load failed: %v", err)
    }

    funcMap := template.FuncMap{
        // 模板中调用：{{ t .Lang "img-compress.hero.title" }}
        "t": i18n.T,
        // 其他已有的模板函数...
    }

    tmpl := template.New("").Funcs(funcMap)
    tmpl = template.Must(tmpl.ParseGlob("templates/**/*.html"))
    tmpl = template.Must(tmpl.ParseGlob("templates/*.html"))
    return tmpl
}
```

### 2.4 Handler 规范（固定写法，复制即用）

```go
func XxxPage(c *gin.Context) {
    lang := c.GetString("lang") // 由 middleware.I18n() 注入

    c.HTML(200, "xxx/xxx.html", gin.H{
        // ✅ 必须透传 Lang，模板中所有 {{ t }} 调用依赖此字段
        "Lang":  lang,
        "Title": i18n.T(lang, "xxx.seo.title"),
        "Desc":  i18n.T(lang, "xxx.seo.desc"),
        // 其他业务数据...
        // 广告配置（如已接入 ad-slot skill）
        "AdsClientID": c.GetString("AdsClientID"),
        "AdsEnabled":  c.GetBool("AdsEnabled"),
    })
}
```

> ⚠️ **禁止事项**
>
> | 写法 | 原因 |
> |------|------|
> | ❌ `"Title": "图片压缩工具"` | 硬编码中文，无法切换语言 |
> | ❌ `lang := "zh"` | 绕过中间件，语言检测失效 |
> | ✅ `lang := c.GetString("lang")` | 唯一允许写法 |

### 3.2 语言切换组件（标准写法）

```html
<!-- 导航栏语言切换 — 标准组件 -->
<div class="lang-switch">
  <a href="?lang=zh"
     class="{{ if eq .Lang "zh" }}active{{ end }}"
     hreflang="zh">
    中文
  </a>
  <span>/</span>
  <a href="?lang=en"
     class="{{ if eq .Lang "en" }}active{{ end }}"
     hreflang="en">
    EN
  </a>
</div>
```

### 3.3 SEO 多语言 head 模板

```html
<head>
  <title>{{ t .Lang "xxx.seo.title" }}</title>
  <meta name="description" content="{{ t .Lang "xxx.seo.desc" }}">

  <!-- hreflang 告知搜索引擎多语言版本 -->
  <link rel="alternate" hreflang="zh" href="{{ .SiteURL }}{{ .Path }}?lang=zh">
  <link rel="alternate" hreflang="en" href="{{ .SiteURL }}{{ .Path }}?lang=en">
  <link rel="alternate" hreflang="x-default" href="{{ .SiteURL }}{{ .Path }}">

  <!-- og 标签同步使用翻译 -->
  <meta property="og:title"       content="{{ t .Lang "xxx.seo.title" }}">
  <meta property="og:description" content="{{ t .Lang "xxx.seo.desc" }}">
  <meta property="og:locale"      content="{{ if eq .Lang "zh" }}zh_CN{{ else }}en_US{{ end }}">
</head>
```

> Handler 需额外透传 `"Path": c.Request.URL.Path` 和 `"SiteURL": cfg.SiteURL`

## 六、常见错误处理

| 错误情况 | 正确处理方式 |
|----------|------------|
| 模板中出现裸中文字符串 | 抽取为 zh.json Key，同步写 en.json，模板改为 `{{ t .Lang "..." }}` |
| zh.json 有 Key 但 en.json 没有 | 补全 en.json 对应 Key，值如不确定可暂填英文占位 |
| `{{ t }}` 输出 Key 本身（未翻译）| 检查 Key 拼写是否与 JSON 文件一致，检查 i18n.Load 是否成功调用 |
| 语言切换后页面文案未变 | 检查路由组是否挂载 middleware.I18n()，Handler 是否透传 Lang |
| Cookie 未写入导致刷新丢失语言 | 检查 middleware 中 SetCookie 的 domain 参数是否匹配当前域名 |
| 新增 Key 热更新不生效 | 翻译文件为启动时加载，修改 JSON 后需重启服务 |
| 多处相同文案需要不同翻译 | 使用更细粒度的 Key 而非共用，例如 `upload.btn` vs `result.download.btn` |

## 七、数据流全景图

```
i18n/zh.json  i18n/en.json
      ↓              ↓
   i18n.Load("i18n")（启动时加载到内存）
              ↓
      middleware.I18n()
        检测语言：?lang= > Cookie > Accept-Language > 默认 zh
        c.Set("lang", "zh" 或 "en")
        写入 Cookie（30天）
              ↓
         Handler
        lang := c.GetString("lang")
        i18n.T(lang, "xxx.seo.title")  → 传给模板 Title/Desc
        gin.H{ "Lang": lang, ... }
              ↓
         HTML 模板
        {{ t .Lang "xxx.hero.title" }}  → i18n.T 查询内存翻译表
        <html lang="{{ .Lang }}">
        <a href="?lang=zh">  <a href="?lang=en">
              ↓
         浏览器渲染
        中文 / English 文案正确显示
        hreflang 告知搜索引擎多语言版本
```

## 八、输出规范

- **输出顺序**：扫描结果（或新 Key 清单）→ zh.json diff → en.json diff → 模板 diff → Handler diff → 验收清单
- **Key 格式**：英文小写 + 连字符，不使用下划线分隔层级（层级用 `.`）
- **翻译质量**：英文翻译需自然流畅，避免逐字直译；中文保持简洁专业
- **改动超过 10 处时**：输出完整替换后的文件，而非 diff
- **说明文字**：与用户当前对话语言一致
