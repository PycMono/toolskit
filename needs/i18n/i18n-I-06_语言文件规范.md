# 语言文件规范与示例

## 文件命名与结构规范

```
规则：
1. 文件名 = 命名空间名，小写，连字符分隔
2. JSON 支持最多 3 层嵌套（loader 会自动展平）
3. 复数 key 使用 { "one": "...", "other": "..." } 子结构
4. 插值使用 Go fmt 格式: %s, %d, %f
5. 所有语言必须有 common.json（基础 key）
```
