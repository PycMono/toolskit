## locales 命名规范

```
locales/{lang-Code}/{namespace}.json

lang-Code 格式: BCP 47
  en-US  英语（美国）
  zh-CN  简体中文
  zh-TW  繁体中文
  ja-JP  日语
  ko-KR  韩语
  fr-FR  法语
  de-DE  德语

namespace 对应业务模块（小写，连字符分隔）:
  common        公共翻译（按钮、操作、状态等）
  auth          认证相关（登录、注册、密码）
  dashboard     仪表盘
  errors        错误信息
  validation    表单验证
  email         邮件模板文本
  [page-name]   具体页面
```
