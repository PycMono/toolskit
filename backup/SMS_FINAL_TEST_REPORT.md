```
1. 购号成功 → currentOrderData 保存
2. 收到 SMS → saveToHistory() 调用
3. 完成订单 → saveToHistory('已使用')
4. 取消订单 → saveToHistory('已取消')
5. 超时订单 → saveToHistory('已超时')
6. 历史页面 → getHistory() 读取
7. 导出 CSV → 遍历 localStorage
8. 清空历史 → localStorage.removeItem()

验证点: 每个操作后 localStorage 数据正确
```

## 🐛 已知问题

### 问题 1: 轮询累积
**描述**: 页面长时间停留，轮询请求累积  
**影响**: 低  
**解决方案**: 页面 `beforeunload` 事件清理轮询  
**状态**: ✅ 已实现（sms-active.js Line 260）

### 问题 2: localStorage 容量
**描述**: 超过 200 条记录后性能下降  
**影响**: 低  
**解决方案**: 自动裁剪（已实现）  
**状态**: ✅ 已实现（sms-history.js Line 38）

### 问题 3: 多标签页同步
**描述**: 多个标签页打开历史页面，数据不同步  
**影响**: 低  
**解决方案**: 监听 `storage` 事件（可选）  
**状态**: ⏸️ 待实现（优先级低）
