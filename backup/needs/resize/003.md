## 3. 验收标准

- [ ] 页面靛蓝主色风格，与压缩工具（翠绿）视觉上明显区分
- [ ] 上传区占满容器，调整大小图标有 hover 动画
- [ ] 拖拽时边框变蓝，背景浅蓝，「松开」提示层出现
- [ ] 上传后选项面板从下方平滑显示
- [ ] 三个模式 Tab 切换正常：像素/百分比/预设面板各自正确显示
- [ ] 像素模式：宽高输入框，锁定按钮默认蓝色（锁定状态），解锁后变灰
- [ ] 百分比模式：滑块拖动，数字实时更新，预算尺寸实时显示
- [ ] 预设模式：点击预设按钮高亮，底部显示选中名称和尺寸
- [ ] 格式四选一，选中蓝色边框，WebP 有「推荐」角标
- [ ] PNG 选中时质量滑块 disabled（PNG 无损）
- [ ] 开始按钮点击后 disabled，文字变「处理中...」
- [ ] 移动端（<768px）预设变两列，特性三列变单列
