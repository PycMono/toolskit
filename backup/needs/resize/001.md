# 多媒体下新增，图片调整大小工具 — 需求代码块总览索引（R-00）

> **竞品对标**：https://webimagetool.com/resize.html  
> **技术方向**：纯前端（Canvas API，无需后端）  
> **路由**：`/img/resize`  
> **主色**：靛蓝 `#2563eb`（与压缩工具翠绿、PDF 工具红色明显区分）

---

## 📐 产品架构图

```
用户拖拽 / 选择图片（JPG / PNG / WebP，最多 20 张）
        ↓
  选择调整模式
  ├─ 像素模式（宽 × 高，可锁定比例）
  ├─ 百分比模式（缩放 %）
  └─ 预设模式（社交媒体尺寸快选）
        ↓
  前端 Canvas 调整引擎（浏览器内完成）
  ├─ 计算目标尺寸
  ├─ Canvas drawImage 绘制
  └─ toBlob 导出（JPEG / PNG / WebP + 质量控制）
        ↓
  结果列表（每张显示：原尺寸 → 新尺寸，文件大小变化）
        ↓
  单张下载 / 全部打包 ZIP 下载
```
