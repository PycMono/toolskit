# PDF 工具套件重构 — 需求代码块总览

> 完全对标 https://pdfhouse.com 开发  
> 涵盖：PDF 编辑、合并、拆分、压缩、转换（双向 12 种格式）、水印、加密、旋转  
> 后端依赖：LibreOffice（格式转换）+ Ghostscript（压缩）+ pdfcpu（PDF 操作）  
> 每个 Block 最小可交付，可独立交给 Copilot 开发

---

## 📐 产品架构图

```
用户
 │
 ├─ 首页 Landing              P-02  工具卡片网格 + 搜索 + 分类 Tab
 │
 ├─ 通用处理流程               P-03  拖拽上传 → 配置参数 → 处理 → 下载（所有工具共用）
 │
 ├─ 核心 PDF 工具
 │    ├─ P-04  合并 PDF        多文件排序合并
 │    ├─ P-05  拆分 PDF        按页范围/固定间隔拆分
 │    ├─ P-06  压缩 PDF        Ghostscript 四级画质压缩
 │    ├─ P-07  PDF 转其他格式   → Word / Excel / PPT / JPG / PNG / TXT / HTML
 │    ├─ P-08  其他格式转 PDF   Word / Excel / PPT / JPG / PNG / TXT → PDF
 │    └─ P-09  PDF 工具集      旋转 / 加水印 / 加密 / 解密 / OCR 文字识别
 │
 └─ 后端基础设施
      └─ P-10  处理引擎         任务队列 + LibreOffice + Ghostscript + pdfcpu + 临时文件清理
```

## 🗓 推荐开发顺序（5天）

```
第 1 天（8h）
  ├─ P-10  后端处理引擎（5h）← 所有工具依赖，必须最先
  └─ P-01  路由+SEO+i18n（2h）+ P-02 首页（1h 骨架）

第 2 天（7h）
  ├─ P-02  首页 Landing 完善（2h）
  └─ P-03  通用处理流程 UI（3h）← 所有工具前端复用此组件

第 3 天（7h）
  ├─ P-04  合并 PDF（4h）← 最复杂，优先级高
  └─ P-06  压缩 PDF（2h）← 最常用

第 4 天（6h）
  ├─ P-05  拆分 PDF（3h）
  └─ P-07  PDF 转出（3h，7 个子工具共用同一模板）

第 5 天（4h）
  ├─ P-08  转入 PDF（2h，6 个子工具共用同一模板）
  └─ P-09  PDF 工具集（2h，旋转/水印/加密）
```

## 🗺 路由规划（完整）

```
# 页面路由
/pdf                    ← 首页 Landing
/pdf/merge              ← 合并 PDF
/pdf/split              ← 拆分 PDF
/pdf/compress           ← 压缩 PDF
/pdf/rotate             ← 旋转 PDF
/pdf/watermark          ← 加水印
/pdf/encrypt            ← PDF 加密
/pdf/decrypt            ← PDF 解密
/pdf/ocr                ← OCR 识别
/pdf/to-word            ← PDF 转 Word
/pdf/to-excel           ← PDF 转 Excel
/pdf/to-ppt             ← PDF 转 PPT
/pdf/to-jpg             ← PDF 转 JPG
/pdf/to-png             ← PDF 转 PNG
/pdf/to-txt             ← PDF 转 TXT
/pdf/to-html            ← PDF 转 HTML
/pdf/from-word          ← Word 转 PDF
/pdf/from-excel         ← Excel 转 PDF
/pdf/from-ppt           ← PPT 转 PDF
/pdf/from-jpg           ← JPG 转 PDF
/pdf/from-png           ← PNG 转 PDF
/pdf/from-txt           ← TXT 转 PDF

# API 路由（后端处理）
POST /api/pdf/upload           ← 上传文件，返回 fileId
POST /api/pdf/process          ← 提交处理任务，返回 taskId
GET  /api/pdf/task/:taskId     ← 查询任务状态（轮询）
GET  /api/pdf/download/:taskId ← 下载结果文件
DELETE /api/pdf/file/:fileId   ← 主动删除临时文件
```
