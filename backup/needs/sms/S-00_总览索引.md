# SMS 接码器重构 — 需求代码块总览

> 完全对标 https://5sim.net 重构“隐私账号/SMS 接码平台” 
> 每个 Block 最小可交付，可独立交给 Copilot 开发  
> 涉及：用户系统、余额系统、实时轮询、5SIM API 代理层、WebSocket 推送

---

## 📐 产品架构图

```
用户
 │
 ├─ 首页（Landing）         S-02  展示功能 + 服务搜索入口
 │
 ├─ 认证系统                S-03  注册 / 登录 / JWT / Google OAuth
 │
 ├─ 用户中心                S-04  余额 / 充值 / API Key / 账户设置
 │
 ├─ 接码主流程              ◀── 核心业务
 │    ├─ S-05  服务选择器（服务 + 国家 + 运营商三级联动）
 │    ├─ S-06  购号 + 等待 SMS（轮询 / WebSocket + OTP 高亮）
 │    ├─ S-07  活跃订单管理（取消 / 完成 / 续租）
 │    └─ S-08  历史订单列表（分页 + 筛选 + 导出）
 │
 ├─ 价格 & 统计             S-09  实时价格表 + 成功率统计
 │
 └─ 后端基础设施
      ├─ S-10  5SIM API 代理层（所有 5sim.net API 请求）
      └─ S-11  WebSocket 实时推送 SMS（服务端推送）
```

## 🗺 路由规划

```
/sms                       ← SMS 接码器首页（Landing）
/sms/buy                   ← 购号主流程（服务选择 + 购号 + 等待）
/sms/active                ← 活跃订单管理
/sms/history               ← 历史订单
/sms/prices                ← 价格 & 统计
/sms/account               ← 用户中心 & 余额
/sms/login                 ← 登录
/sms/register              ← 注册

API 路由（后端代理层）：
/api/sms/products          ← 服务列表（代理 5sim）
/api/sms/countries         ← 国家列表
/api/sms/operators         ← 运营商 + 价格
/api/sms/buy               ← 购买号码
/api/sms/order/:id         ← 订单状态
/api/sms/order/:id/cancel  ← 取消订单
/api/sms/order/:id/finish  ← 完成订单
/api/sms/order/:id/rebuy   ← 重新购买
/api/sms/orders/active     ← 活跃订单
/api/sms/orders/history    ← 历史订单
/api/sms/prices            ← 价格统计
/ws/sms/:orderId           ← WebSocket SMS 推送
```
