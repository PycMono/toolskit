# 图片压缩工具 — 需求代码块总览索引

> 竞品对标：https://tinypng.com  
> 技术方向：**纯前端压缩**（Canvas API + browser-image-compression + JSZip）  
> 无需后端接口 / 无需用户登录  
> 图片全程不离开用户浏览器，零隐私风险

---

## 📐 产品架构图

```
用户拖拽 / 选择图片
        ↓
  格式识别（JPEG / PNG / WebP / GIF）
        ↓
  前端压缩引擎（浏览器内完成）
  ├─ JPEG/WebP → Canvas toBlob(quality)
  ├─ PNG       → browser-image-compression（WASM）
  └─ GIF       → 提示"不支持压缩"
        ↓
  结果列表（每张显示：原始大小 → 压缩后 → 节省率）
        ↓
  单张下载 / 全部打包 ZIP 下载
```

## 📌 设计风格定调（对标 TinyPNG）

- **色调**：TinyPNG 用橙色，本次用 **翠绿（#1a9b6c）** 作为主色，与 PDF 工具（红）/ SMS 工具（紫）形成区分
- **背景**：浅米白 `#fafaf8`，卡片白色，边框 `#e8e4dc`
- **字体**：保持站内统一
- **上传区**：占满视口中央，大虚线边框，拖拽时背景变绿
- **结果列表**：每张图片一行，左缩略图 + 中间进度条 + 右大小对比 + 下载按钮
- **差异化亮点**：质量滑块（20%~100%）、格式转换（→ WebP/JPG/PNG）、Before/After 图片滑块对比
