# JSON 工具集 — 需求代码块总览索引

> 竞品对标：https://jsonlint.com  
> 技术方向：**纯前端处理**（Monaco Editor + 原生 JS 处理逻辑）  
> 无需后端接口（代码生成、格式转换、解析全部前端完成）  
> 唯一后端：Token Counter（需调用模型 tokenizer API）  
> 主色：**琥珀橙 #f59e0b**，与 PDF（红）/ 图片压缩（绿）/ SMS（紫）区分

---

## 📐 产品架构图

```
用户输入 JSON 文本（粘贴 / 上传 / 示例）
        ↓
  前端解析引擎（JSON.parse / 自定义 parser）
  ├─ 验证 & 格式化 → Validate / Pretty Print / Minify / Sort / Escape...
  ├─ 可视化查看   → Tree Viewer / Table Viewer
  ├─ 对比分析    → Diff / Path / Search / Size / Flatten
  ├─ 数据转换    → CSV / YAML / XML / SQL / Markdown / Excel
  ├─ 代码生成    → TypeScript / Python / Java / C# / Go / Kotlin / Swift / Rust / PHP
  ├─ Schema 工具 → Schema Validate / Schema Generate
  └─ 编码工具    → Base64 / JWT / JSONC / Token Count
        ↓
  结果输出（编辑器展示 / 下载文件 / 复制到剪贴板）
```

## 🎨 设计风格定调

- **主色**：琥珀橙 `#f59e0b`，hover 深色 `#d97706`，浅色背景 `#fffbeb`
- **背景**：浅米白 `#fafaf8`，卡片白色，边框 `#e8e4dc`
- **编辑器布局**：左右双栏（输入 | 输出），中间操作按钮栏
- **移动端**：上下堆叠，输入在上输出在下
- **公共组件**：`JT`（JSON Tools）命名空间前缀，统一 CSS 变量
