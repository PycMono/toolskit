# 模板集成

## internal/template/funcmap.go

```go
package template

import (
	"html/template"

	"yourproject/internal/i18n"
)

// BuildFuncMap 构建模板函数 map
// 注意：Gin 的 FuncMap 在加载模板时固定，动态值通过传入 *Translator 实现
func BuildFuncMap(t *i18n.Translator) template.FuncMap {
	return template.FuncMap{
		// 直接翻译（通过模板 data 中的 T 对象调用）
		// 在模板中通过 .T.T "key" 调用
		// 以下是全局 helper，可选
		"safeHTML": func(s string) template.HTML {
			return template.HTML(s)
		},
		"dict": func(values ...interface{}) map[string]interface{} {
			d := make(map[string]interface{})
			for i := 0; i+1 < len(values); i += 2 {
				key, _ := values[i].(string)
				d[key] = values[i+1]
			}
			return d
		},
	}
}
```

## 模板中的调用方式

由于 Gin 的 `html/template` 不支持运行时动态 FuncMap，
推荐将 `*Translator` 作为 **模板数据** 传入，
在模板中通过方法调用使用。

### templates/layout/base.html

```html
<!DOCTYPE html>
<html lang="{{ .Lang }}" {{ if .IsRTL }}dir="rtl" {{ end }}>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>{{ .T.T "common.site.title" }}</title>
</head>
<body>
<nav>
    <a href="/">{{ .T.T "common.nav.home" }}</a>
    <a href="/dashboard">{{ .T.T "common.nav.dashboard" }}</a>

    <!-- 语言切换器 -->
    <div class="lang-switcher">
        <a href="?lang=en-US">English</a>
        <a href="?lang=zh-CN">中文</a>
        <a href="?lang=ja-JP">日本語</a>
    </div>
</nav>

{{ block "content" . }}{{ end }}

</body>
</html>
```

### templates/auth/login.html

```html
{{ template "layout/base.html" . }}

{{ define "content" }}
<div class="auth-container">
    <h1>{{ .T.T "auth.login.title" }}</h1>

    <form method="POST" action="/login">
        <div class="form-group">
            <label>{{ .T.T "auth.login.email_label" }}</label>
            <input type="email" name="email"
                   placeholder="{{ .T.T "auth.login.email_placeholder" }}">
        </div>

        <div class="form-group">
            <label>{{ .T.T "auth.login.password_label" }}</label>
            <input type="password" name="password">
        </div>

        <button type="submit">
            {{ .T.T "auth.login.submit_button" }}
        </button>
    </form>

    <!-- 插值示例 -->
    <p>{{ .T.TF "auth.login.welcome" .Username }}</p>

    <!-- 复数示例 -->
    <p>{{ .T.TN "common.notification.count" .NotificationCount }}</p>

    <a href="/register">{{ .T.T "auth.login.no_account" }}</a>
</div>
{{ end }}
```

## 错误信息翻译（在 Handler 中使用）

```go
func registerHandler(c *gin.Context) {
    t := middleware.MustTranslator(c)

    var form RegisterForm
    if err := c.ShouldBind(&form); err != nil {
        c.HTML(http.StatusBadRequest, "auth/register.html", gin.H{
            "T":     t,
            "Lang":  t.Lang(),
            "Error": t.T("errors.form.invalid"),
        })
        return
    }

    // 业务逻辑错误翻译
    if userExists {
        errMsg := t.TF("auth.register.error.email_exists", form.Email)
        // ...
    }
}
```
