## 🚀 使用流程

1. 访问 `/img/resize` 或 `/media/image-resize`
2. 拖拽或选择图片（最多 20 张，单张最大 10MB）
3. 选择模式：
   - **像素**：输入宽高（可锁定比例）
   - **百分比**：输入缩放百分比（50% = 缩小一半）
   - **预设**：一键选择 Instagram/Facebook/YouTube 等尺寸
4. 设置输出格式（保持原格式 / 转为 JPG/PNG/WebP）
5. 调整质量（60~100%，仅 JPG/WebP 有效）
6. 点击"开始调整大小"
7. 实时查看结果列表（原尺寸 → 新尺寸，文件大小变化）
8. 单张下载 或 全部打包 ZIP 下载
9. 点击"预览"查看 Before/After 对比
