## 2. 卡片 & Drawer UI 规范

（详细规范见 I-02 CSS 规范及 I-03 buildCard / openDrawer 函数）

**卡片布局（三区）：**
```
┌──────────────────────────────────┐
│ [分类角标]           [记录数]     │  ← Header 行
│ 数据集名称（H3）                  │  ← 主标题
│ 描述文字（2行截断）               │  ← 描述
│ Fields: code, name, city +3 more │  ← 字段预览
├──────────────────────────────────┤
│ [12 KB]  [Copy] [Validate] [↓]  │  ← Footer 操作区
└──────────────────────────────────┘
```

**Drawer 字段类型标签颜色：**

| 类型 | 颜色 | class |
|------|------|-------|
| string | 绿色 `#22c55e` | `ds-type--string` |
| number | 蓝色 `#3b82f6` | `ds-type--number` |
| boolean | 紫色 `#a855f7` | `ds-type--boolean` |
| array | 橙色 `#f97316` | `ds-type--array` |
| object | 青色 `#06b6d4` | `ds-type--object` |
| null | 灰色 `#6b7280` | `ds-type--null` |
